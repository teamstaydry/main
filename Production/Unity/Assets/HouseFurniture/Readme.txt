Thank you for the purchase! By default the asset made using build in pipeline

To change the SRP please follow the intruction

HDRP
Add HDRP from package manager
Unzip HDRP_Pack by double click it, press import
Press fix all in the toolbar it will ask for Scriptable Render Asset, select it
Load Demo_HDRP scene

For version 2022+ after HDRP_Pack please import Shaders from HDRP_2022 same if you use URP

URP
Add URP from package manager
Unzip URP_Pack by double click it, press import
Assign Scriptable Render Asset
Load Demo_URP scene
